Checking web compatibility

When working with web content, it is important to check what browsers your content is compatible with.

There are three ways to do this:

caniuse provides a basic compatibility table for most major desktop and mobile browsers.

MDN goes a step further than Caniuse, it also provides additional information about the property, how it works and the specification that the property belongs to.

Both Caniuse and MDN require you to have a browser open on the right site and actively search for the property or API youâ€™re looking for.
