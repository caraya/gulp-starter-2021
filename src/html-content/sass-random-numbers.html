Generating randon numbers with Sass

I've been working on a crazy idea. Create content that is randomly positioned in the page and then rotated a random percentage between a negative and a positive number. Something similar to what Dudley Storey did in Random Stacked Images with the Web Animation API and Progressive JS but using as little Javascript as possible and without animation, just positioning the images in a random position.

The rough shape of the SCSS looks like this:

.post-it {
  top: random(400) - 100 + px;
  left: random(400) - 100 + px;
  transform: rotate(random(90) - 45 + deg);
  width: random(400) - 100 + px;
}
